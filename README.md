# SQL Avanzado – Proyecto de Práctica

Proyecto personal enfocado en la práctica avanzada de SQL utilizando PostgreSQL.

El objetivo es desarrollar consultas complejas, aplicar buenas prácticas de modelado relacional y documentar soluciones tipo portfolio técnico.

---

## 🗂 Estructura del Proyecto

### 01_schema
Definición de tablas y restricciones:
- create_tables.sql
- constraints.sql

### 02_data
Carga de datos:
- Dimensiones
- Tabla de hechos
- Datos intencionadamente incorrectos para ejercicios de calidad

### 03_data_quality
Validaciones y detección de errores:
- Foreign keys inválidas
- Problemas de formato
- NULLs inesperados

### 04_queries
Ejercicios técnicos:
- JOINs
- CTE
- Window functions
- Ranking
- Running totals
- Subqueries

### 05_analisis
Consultas orientadas a negocio:
- Ventas por mes
- Top categorías
- KPIs generales
- Análisis por país

---

## 🎯 Objetivo

Practicar SQL avanzado:
- Modelado relacional
- Integridad referencial
- CTE
- Window functions
- Agregaciones avanzadas
- Limpieza de datos
- Análisis exploratorio
